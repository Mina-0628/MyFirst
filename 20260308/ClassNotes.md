# 使用 VSCode 編寫 HTML

## Step1：安裝軟體 VSCode（編寫程式的軟體）

下載並安裝 VSCode：

https://code.visualstudio.com/

用途：  
用來編寫 **HTML / CSS / JavaScript**。

---

# Step2：安裝 VSCode 擴充功能

在 VSCode 的 **Extensions（擴充功能）** 搜尋並安裝以下插件：

- Live Server  
- Auto Close Tag  
- Auto Rename Tag  
- cdnjs  
- Chinese Lorem  
- fake image snippet collection  
- Git Graph  
- HTMLHint  
- Image preview（小圖示，可安裝或不安裝）  
- prettier - code formatter  
- SCSS Everywhere  
- GitHub Copilot Chat  
- Windsurf Plugin  
- Path Intellisense  

用途：  
讓 VSCode 在寫 **HTML / CSS / JavaScript** 時更方便。

---

# Step3：將電腦中的檔案上傳到 GitHub（雲端）存放

使用 **Git** 進行版本控制，  
並將本地端（電腦）的檔案上傳到 **GitHub** 做雲端儲存與管理。

---

## 如果不會使用 Git / GitHub

可以直接下載老師提供的壓縮檔：

https://github.com/summer10920/studiesAdvanced/archive/refs/heads/11501/rwd.zip

2. 解壓縮
